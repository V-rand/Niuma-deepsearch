查询 Wikipedia 获取人物、事件、作品等结构化信息。使用 Wikipedia REST API（单次请求），速度快。

参数：
- query: 页面标题或搜索关键词（精确标题返回最准确）
- lang: 语言代码 "zh" 或 "en"（默认 en）

返回：
- title: Wikipedia 页面标题
- url: 页面链接
- summary: 首段摘要（约 200-2000 字）
- description: 页面简短描述（如 "1880–1881 war in South Africa"）
- pageid: Wikipedia 页面 ID
- error: 不可用时为 "unavailable" 或 "not_found"

适用于：演员出生地、剧集首播年份、人物生平、事件日期等确切事实查询。
不适用于：需要最新资讯、深度分析的场景。

注意：中国大陆访问 Wikipedia 可能较慢，查询超时时请改用 web_search。
