读取文件内容。如果文件不存在则返回错误。

参数：
- path: 文件路径（相对工作目录）
- offset: 起始行号（从0计数，默认0）。不指定 offset 时从文件开头读
- limit: 最大返回行数。不指定时返回全部行。大文件建议分多次读取，每次 500-1000 行
- encoding: 文件编码，默认 utf-8

返回：
- path: 文件路径
- content: 文件内容片段
- size: 总字符数
- total_lines: 文件总行数
- offset/limit/displayed_lines: 分段读取时的定位信息

注意：
- 二进制文件（图片、PDF 等）会报错，先用 upload_parse 解析
- 返回内容中的行号和原始文件一致，不是从 offset 重新计数
- 可以和其他 file_* 工具并行调用
