# 中国民事案件生命周期阶段定义（面向 AgentOS）

## 说明
本文件把案件生命周期按“阶段”结构化，便于 AgentOS 为每个 session 建模。

每个阶段描述：
- 目标
- 输入
- 核心动作
- 输出
- 常见分支
- 风险点
- 退出条件

---

## 1. 委托代理阶段

### 目标
建立案件初始上下文，判断是否承接，建立服务关系。

### 输入
- 客户初步陈述
- 客户基础身份信息
- 初始案件材料
- 客户法律服务需求

### 核心动作
- 了解基本案情
- 评估收案风险
- 利益冲突检索
- 收集和整理材料
- 起草谈话笔录、材料清单、策略分析报告
- 向客户反馈并修订策略
- 办理委托手续
- 收取代理费
- 确定律师团队并建立服务群

### 输出
- 谈话笔录
- 利益冲突检索报告
- 材料清单 / 补充材料清单
- 诉讼策略分析报告
- 委托代理合同
- 授权委托书
- 风险告知书

### 常见分支
- 收案
- 不收案
- 客户继续补充材料后再决策

### 风险点
- 材料不全
- 利益冲突
- 风险未充分揭示
- 委托手续不完整

### 退出条件
- 已决定是否代理
- 若代理，已形成完整启动资料并建立案件 session

---

## 2. 案件分析阶段

### 目标
把案件转化为“事实-证据-法律-策略”结构。

### 输入
- 委托阶段积累的材料
- 客户补充材料
- 初始事实框架

### 核心动作
- 继续收集证据
- 检索法律条文、案例、学术文章
- 团队分析
- 识别证据薄弱点
- 补强证据
- 制定诉讼方案

### 输出
- 证据清单
- 法律数据检索分析报告
- 诉讼方案
- 补强材料清单

### 常见分支
- 进入诉前保全
- 直接进入立案 / 应诉
- 因材料不足继续补证

### 风险点
- 法律路径判断错误
- 事实结构不清
- 证据不足

### 退出条件
- 已形成可执行诉讼策略

---

## 3. 诉前保全阶段（可选）

### 目标
在起诉前通过保全锁定财产或固定现状。

### 输入
- 已形成的诉讼主张
- 财产线索
- 担保方案
- 起诉准备材料

### 核心动作
- 起草保全申请
- 向法院递交保全材料
- 提供担保
- 跟进法院裁定和实施
- 必要时复议或申请解除保全
- 在 30 日内起诉

### 输出
- 诉前财产/行为保全申请书
- 担保书
- 保全情况记录
- 复议申请书（如有）
- 解除财产保全担保申请书（如有）

### 常见分支
- 准予保全
- 不予保全
- 被申请人提供担保
- 30 日内未起诉导致解除保全

### 风险点
- 财产线索不足
- 担保方案不可行
- 未及时起诉导致保全失效

### 退出条件
- 保全结束并转入正式诉讼，或决定不再推进保全

---

## 4. 立案 / 应诉阶段

### 目标
完成立案或答辩，建立正式程序地位。

### 输入
- 起诉状或起诉状副本
- 证据材料
- 主体资格材料
- 委托代理手续

### 核心动作
- 起草起诉状 / 答辩状
- 提交立案材料或答辩材料
- 提交证据目录和证据材料
- 缴纳诉讼费
- 跟进立案结果
- 提出保全、调查令、延期举证、回避、管辖异议等申请

### 输出
- 起诉状 / 答辩状
- 证据目录
- 程序申请书
- 立案情况汇报 / 答辩情况汇报

### 常见分支
- 成功立案
- 裁定不予受理
- 提出上诉
- 提出程序性申请

### 风险点
- 超过答辩期限
- 未按时缴费
- 举证不完整
- 材料份数或格式错误

### 退出条件
- 程序启动完成，进入庭前准备

---

## 5. 庭前准备阶段

### 目标
将案件材料转化为庭审能力。

### 输入
- 立案后材料
- 双方证据
- 法院通知
- 检索资料

### 核心动作
- 案情研讨
- 类案检索
- 制作庭审大纲
- 模拟法庭（可选）
- 庭审告知与协调
- 确认证据原件
- 统计出庭人员
- 制作交通预案
- 准备和解预案

### 输出
- 案例检索报告
- 庭审大纲
- 模拟法庭复盘报告
- 客户参加庭审情况统计表
- 交通预案
- 和解协议草案（如有）

### 常见分支
- 调解前置
- 正常开庭
- 需要追加补证

### 风险点
- 原件未准备
- 客户沟通不到位
- 庭审焦点不清晰

### 退出条件
- 开庭准备充分并收到明确庭审安排

---

## 6. 庭审阶段

### 目标
在法庭上完成事实、证据和法律意见的表达与攻防。

### 输入
- 庭审大纲
- 证据原件
- 质证意见
- 代理意见提纲

### 核心动作
- 出庭
- 核对原件
- 质证和辩论
- 回应法官和对方观点
- 保存庭审记录
- 调解时审核协议

### 输出
- 庭审情况简报
- 补充提交材料清单
- 调解协议（如达成）

### 常见分支
- 调解成功
- 调解失败继续庭审
- 庭后补证
- 等待判决

### 风险点
- 原件缺失
- 关键事实未表达清楚
- 调解协议风险

### 退出条件
- 当次庭审结束并完成庭后转写

---

## 7. 庭后阶段

### 目标
延续庭审成果并跟进裁判结果。

### 输入
- 庭审记录
- 法官要求
- 庭后补充材料需求

### 核心动作
- 庭审汇报
- 补充证据
- 提交代理词 / 补充意见
- 与法官沟通进展
- 向客户同步结果
- 提示上诉期限
- 全案复盘

### 输出
- 代理词
- 补充证据清单
- 全案复盘报告
- 裁判结果通知

### 常见分支
- 判决
- 调解书
- 裁定
- 需上诉或执行

### 风险点
- 错过上诉期限
- 补证不及时
- 客户对结果预期失控

### 退出条件
- 裁判生效或决定上诉

---

## 8. 履行 / 执行阶段

### 目标
推动裁判结果落地。

### 输入
- 生效裁判文书
- 履行期限信息
- 财产线索

### 核心动作
- 督促自动履行
- 询问客户是否申请执行
- 起草执行申请书
- 执行立案
- 联系执行法官
- 财产普查
- 推动执行措施
- 落实威慑措施
- 执行和解
- 执行异议

### 输出
- 执行申请书
- 立案情况汇报书
- 财产普查记录
- 执行和解协议
- 执行异议申请书

### 常见分支
- 自动履行
- 进入执行
- 达成执行和解
- 执行完毕
- 终结执行

### 风险点
- 超过申请执行期限
- 财产线索不足
- 执行推进不及时

### 退出条件
- 执行完结或不再有可执行动作

---

## 9. 上诉前准备阶段

### 目标
复盘一审并形成二审策略。

### 输入
- 一审卷宗
- 一审裁判文书
- 一审庭审与证据材料

### 核心动作
- 调取一审卷宗
- 解构裁判文书
- 复盘当事人主体、诉辩主张、请求权基础和法院论证
- 分析一审问题
- 制定二审诉讼策略
- 向客户反馈并修订

### 输出
- 一审裁判文书结构报告
- 案例检索报告
- 诉讼策略报告
- 证据清单

### 常见分支
- 决定上诉
- 决定不上诉

### 风险点
- 错过上诉期限
- 只重复一审思路而未识别裁判问题

### 退出条件
- 已形成明确二审行动方案

---

## 10. 二审立案 / 答辩阶段

### 目标
完成二审程序启动并跟进案件移送。

### 输入
- 上诉状 / 答辩状
- 一审裁判文书
- 身份与授权材料

### 核心动作
- 提交上诉材料或答辩材料
- 缴纳上诉费
- 跟进案件移送
- 检索法律和案例
- 准备二审庭前材料

### 输出
- 上诉状 / 答辩状
- 二审立案进度记录

### 退出条件
- 二审法院受理并进入准备或开庭阶段

---

## 11. 二审庭审阶段

### 目标
围绕一审问题进行二次表达和纠错。

### 输入
- 二审策略
- 二审庭审大纲
- 原件和补充材料

### 核心动作
- 参加二审庭审
- 提交补充材料
- 提交代理词
- 汇报庭审情况

### 输出
- 庭审情况简报
- 代理词
- 补充材料清单

### 退出条件
- 二审裁判作出并进入执行或再审判断

---

## 12. 二审庭后及执行阶段

### 目标
接收二审裁判并处理后续执行或再审问题。

### 输入
- 二审裁判文书
- 生效时间
- 客户决策

### 核心动作
- 签收裁判文书
- 汇报结果
- 提示再审期限
- 进入履行 / 执行
- 结案归档

### 输出
- 结果通知
- 再审告知
- 结案小结

### 退出条件
- 履行/执行完毕或转入再审/抗诉

---

## 13. 再审 / 抗诉阶段

### 目标
对生效裁判寻求特殊救济。

### 输入
- 生效裁判
- 再审理由
- 再审申请材料

### 核心动作
- 提交再审申请
- 向对方送达副本
- 跟踪法院审查
- 必要时寻求检察建议或抗诉

### 输出
- 再审申请书
- 再审进展记录

### 退出条件
- 再审被驳回、启动再审，或完成抗诉路径判断

---

## 14. 结案与归档阶段

### 目标
完成案件收尾和知识沉淀。

### 输入
- 全案卷宗
- 原件交接情况
- 最终结果

### 核心动作
- 结案小结
- 原件归还
- 卷宗移送
- 扫描归档
- 装卷
- 内部审批

### 输出
- 结案小结
- 装卷目录
- 归档卷宗

### 退出条件
- session 进入 closed 状态
